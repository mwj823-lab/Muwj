## 1. About the Project
This is my C++ mini-project. It is a small "Detective Game" where the player has to solve a mystery, I wanted to make something different from a basic database, so I created a game where you collect clues and interrogate people to find the killer.

## 2. How to Play
- When you start, you will see a list of choices.
- You can talk to the suspects to hear their stories.
- You can search the room to find hidden evidence.
- Once you are sure, you can accuse the person you think is guilty.

## 3. C++ Concepts Used
I applied what we learned in class to build this project:
* "Struct:" I used a struct called `Person` to group the name, their talk, and their suspicion level together.
* "Arrays:" I stored the 3 suspects in an array to make it easy to loop through them.
* "While Loop:" The game stays running using a `while` loop until the player decides to exit or solve the case.
* "Switch Case:" I used this to handle the main menu choices (1, 2, 3, or 4).
* "Functions:" I made a separate function for the welcome message to keep the `main` code clean.

## 4. My Special Feature (Creativity)
I added a "Suspicion Level" system. When you find a clue, the code automatically updates the suspicion percentage for the suspect. This makes the game more dynamic and fun to play.
